\section*{Question 1}
Using the graphic method of solution, find the optimal solution for the following LP model:

Minimize:
\begin{equation*}
	\text{Z} = 2200 -7\text{X}_1 - 2\text{X}_2
\end{equation*}
Subject to:
\begin{gather}
	\text{X}_1 <= 100 \\
	3\text{X}_1 + 5\text{X}_2 <= 900 \\
	4\text{X}_1 + 2\text{X}_2 <= 600 \\
	\text{X}_1 >= 0, \text{X}_2 >= 0 \nonumber
\end{gather}

The graphical solution to this problem is found in Figure~\ref{fig:q1}. The optimal value for Z is approximately $184.857$, when X$_1 = 85.714$ and X$_2 = 128.571$.
